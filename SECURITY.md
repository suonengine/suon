# Security Policy

## Supported versions

Suon is under active development. Security fixes are applied on a best-effort basis to the latest development state of the repository.

## Reporting a vulnerability

Please do not open public issues for suspected security vulnerabilities.

Instead, report them privately to `ramonbernardo.contato@gmail.com` with:

- A clear description of the issue
- Steps to reproduce or a proof of concept, if available
- Impact assessment
- Any suggested mitigation or fix

We will acknowledge receipt as soon as possible, investigate the report, and coordinate next steps before any public disclosure when appropriate.

## Disclosure

Please give maintainers a reasonable amount of time to investigate and ship a fix before publicly disclosing a vulnerability.
